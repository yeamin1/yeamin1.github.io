ğŸ“ Image auto-scaling with ruler detection

This project aims to develop an automated way to access the image scale in pixel per millimeter (pp/mm) -- by counting how many pixel within in the gap between two tick marks in a ruler with millimeter unit. I solve this problem by:

  • Fine-tuning the Segment Anything Model (SAM)
  • Using a computer vision method to locate the tick marks on a ruler
  • Made an iteratively ruler line detection algorithm for computing the pixel distance within a set of tick marks.

Some outcomes/highlights:

  • The fine-tuned SAM achieves sufficient segmentation performance \(\text{IOU} = 0.89\)
  • The experimental result shows an acceptable ~3% error rate
  • Algorithm has been published to a scientific journal
